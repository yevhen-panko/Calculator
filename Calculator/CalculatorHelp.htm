Calculator help

The calculator can be used to perform simple operations such as addition, subtraction, multiplication, and division. The calculator also provides the possibility to perform a calculation functions . You can perform calculations by clicking the calculator buttons, or by entering the characters from the keyboard. In addition, the available input numbers and operations on the numeric keypad when you press NUM LOCK.

You can use next functions in your expressions:

  • max(a, b, c)
  • min(a, b, c)
  • sum(a, b, c)
  • abs(a)
  • sqrt(a)
  • pi()

Max function

Max function return maximum value from arguments in brackets. Minimal count of arguments is 2. Operator separator is ','. If the count of operators is less than 2, programm give message about error, and perform will be stopped.

For example:

max(100, 300, 200) give result 300.

Min function

Min function return minimum value from arguments in brackets. Minimal count of arguments is 2. Operator separator is ','. If the count of operators is less than 2, programm give message about error, and perform will be stopped.

For example:

min(100, 300, 200) give result 100.

Sum function

Sum function return result of addition all operators in brackets. Minimal count of operators is 2. Operator separator is ','. If the count of operators is less than 2, programm give message about error, and perform will be stopped.

For example:

sum(100, 200, 300) give result 100 + 200 + 300 = 600.

Abs function

Abs function return absolube value of argument in brackets. Count of arguments is 1. If the count of operators is less or more than 1, programm give message about error, and perform will be stopped.

For example:

abs(-100) give result 100.

Sqrt function

Sqrt function return square root value of argument in brackets. Count of arguments is 1. If the count of operators is less or more than 1, programm give message about error, and perform will be stopped. If argument is negative, programm give message about error, and perform will be stopped.

For example:

sqrt(100) give result 10.

Pi function

Pi function return PI value. Count of arguments is 0. If the count of operators is more than 0, programm give message about error, and perform will be stopped.

For example:

pi() give result 3.1415926536.

You also can change theme in Menu Theme. Now is two themes: Orange and Blue. Menu item copy - copy value from text field to system clipboard, and menu item paste - paste value from clipboard to text field. Menu item exit - closing application.
